# 玄历

一个纯前端的历法原型，支持：

- 阳历 / 农历 / 干支四柱
- 生肖与生肖图标（按立春换岁）
- 节气
- 当前或指定日期的时柱
- 西方星座与图标
- 值年卦（可点开看卦象、卦辞、彖辞、象辞）
- 公元 / 公元前年份切换
- 朝代标示（从汉朝开始）
- 帝王 / 年号纪年（可点开帝王详情）
- 农历闰月与农历月份结构展示

## 运行

```bash
python3 -m http.server 8000
```

然后访问 `http://localhost:8000`

## 当前规则

- 年柱：按 `立春` 切换
- 月柱：按 `节气` 切换
- 日柱 / 时柱：按 `23:00` 后子初换日
- 农历：使用浏览器内建 `Intl` 中文农历能力
- 值年卦：按 `皇极经世` 的会 / 运 / 世 / 年序列推算
- 朝代：做展示型映射，从 `西汉` 起标示
- 帝王纪年：按中原主线展示皇帝与年号；

## 文件

- `index.html`：页面结构
- `styles.css`：界面样式
- `calendar-core.js`：历法、节气、干支、值年卦、朝代逻辑
- `app.js`：交互与渲染

## 说明

古代年份部分采用现代公历、现代农历规则回推，用来做展示和研究没问题，但它不等同于逐朝代的官方实历复原。

卦辞 / 彖辞 / 象辞 当前优先从外部公开文本源按需加载；如果外部源不可用，页面会退回到本地卦意摘要。
